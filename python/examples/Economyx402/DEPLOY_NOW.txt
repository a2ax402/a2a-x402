🚀 DEPLOY YOUR CONTRACT TO BSC - STEP BY STEP
═══════════════════════════════════════════════════════════════

IMMEDIATE STEPS TO DEPLOY:

1. SET YOUR PRIVATE KEY
   
   Windows PowerShell:
   $env:PRIVATE_KEY="0xYourPrivateKeyHere"
   
   Linux/Mac:
   export PRIVATE_KEY=0xYourPrivateKeyHere

2. GET BNB FOR GAS (Get test BNB first!)
   
   Testnet Faucet: https://testnet.bnbchain.org/faucet-smart
   - Connect your wallet
   - Request test BNB
   - You need at least 0.001 test BNB

3. INSTALL DEPENDENCIES
   
   pip install web3 eth-account

4. RUN DEPLOYMENT
   
   cd python/examples/bsc-demo/contracts
   python deploy_now.py testnet
   
   For mainnet:
   python deploy_now.py mainnet
